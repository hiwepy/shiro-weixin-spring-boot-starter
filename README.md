# shiro-weixin-spring-boot-starter

### 说明


 > 基于 Shiro 的 Spring Boot Starter 实现

1. 实现基于短消息的注册、登录

### Maven

``` xml
<dependency>
	<groupId>${project.groupId}</groupId>
	<artifactId>shiro-weixin-spring-boot-starter</artifactId>
	<version>2.0.0.RELEASE</version>
</dependency>
```
